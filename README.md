# InventariosCTPSantoDomingo
Sistema de control de activos para el Colegio Tecnico Profesional de Santo Domingo de Heredia
Este sitema esta creado con 3 capas BLL,DAL, UI.
Creado en SQL Server.
Descripcion: Este sistema fue diseñado para el Colegio Tecnico Profesional de Santo Domingo de Heredia,cuenta con la adminitracion de 
activos, reportes, seguimientos,reportes de averias.
